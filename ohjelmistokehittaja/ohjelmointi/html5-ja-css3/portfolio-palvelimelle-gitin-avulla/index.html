Määritellään Git

  1. 1. Asenna kannettavallesi "Git"
  2. 2. Käynnistä Windiws Powershell
  3. 3. Mene kansioon C:/tiedotot/portfolio
  4. 4. Anna komento
    git init
    (tämä määrittelee git maisterin)
  5. 5. Mene kansioon O:\intra
  6. 6. Anna komento
    git clone c:\tiedostot\portfolio
    (tällä kloonataan C-levyaseman kansio O-levyasemalle)

Muutoksien hoitaminen C-levyasemalta O-levyasemalle

  1. Lisää kansioon c:\tiedostot\portfolio alikansiot "css" ja "images"
  2. Lisää portfoli-kansioon tiedosto index.html, css-kansioon tiedostot layout.css ja reset.css ja talenna kansioon images koulun logo
  3. Käynnistä Powershell
  4. Mene kansioon C:/tiedotot/portfolio
  5. Anna komento
    git add .
    (lisätään kaikki muutokset git-stagelle)
  6. Anna komento
    git commit -m "Juuri valmis"
    (Hyväksytään kaikki stagelle siirretyt muutokset)
  7. Mene kansioin O:\intra\portfolio
  8. Anna komento
    git pull c:\tiedostot\portfolio
    (git tuo kaikki muutokset C-levyn kansiosta O-levyn kansioon.
